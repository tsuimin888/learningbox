# 任务清单管理应用

一个基于tkintertools（desgined by Xiaokang2022)和Lingma大模型的低代码开发桌面任务清单管理应用。

## 功能特性

- 创建和管理任务
- 设置任务截止日期和重要程度
- 番茄工作法计时器
- 主题切换（深色/浅色）
- 数据持久化存储
